# Automatic and Manual Image Slider

A simple image slider built with HTML, CSS, and JavaScript.

## Features

- Automatic slide change every 5 seconds
- Manual navigation using radio-button controls
- Smooth transition animation between slides
- Clean and minimal UI

## Project Structure

- `Automatic and Manual Image Slider in HTML CSS.html` - Main slider implementation (HTML + CSS + JS)

## How to Run

1. Open `Automatic and Manual Image Slider in HTML CSS.html` in any modern web browser.
2. Watch the slider auto-advance.
3. Use the manual controls (small buttons) to switch slides directly.

## Customization

- Change image URLs in the `<img src="...">` tags to use your own images.
- Update slider size in the `.slider` and `.slide img` CSS rules.
- Adjust auto-slide timing in JavaScript:

```js
setInterval(..., 5000);
```

Change `5000` to another value (in milliseconds) as needed.
